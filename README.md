# pierogi

> Alternative npm client with a focus on discoverability


## Install

```shell
$ npm install --global pierogi
```


## Usage

Right now `pierogi` only does search.

```shell
$ pier some search terms
```

Basically `pier foo` is the same as `npm search foo`.
